Site do Atelier

Este projeto fictício foi desenvolvido como uma Landing Page para um atelier especializado em bolos, doces, salgados e kits para festas. O objetivo principal foi demonstrar minhas habilidades no desenvolvimento de interfaces web utilizando HTML5 e CSS3, com foco em estruturação semântica do conteúdo, design responsivo para diferentes dispositivos, criação de um menu interativo e intuitivo, organização visual por seções, destacando os produtos de forma clara e atraente. O site conta com uma vitrine de produtos, links de navegação rápidos e área para contatos e redes sociais. O layout busca oferecer uma experiência simples e agradável, ressaltando a identidade visual do atelier fictício.
