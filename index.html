Meus Projetos

☕ Site do Café

Projeto fictício criado para representar a identidade de uma cafeteria artesanal com três blends exclusivos: Cappuccino, Espresso e Mocha. Desenvolvido com HTML5 semântico e CSS3 responsivo, o layout se adapta de forma fluida a desktop e mobile, incluindo menu hambúrguer para telas menores. O site conta com seções animadas com transições suaves, cards de produto com efeitos de escala e sombra ao hover, e um formulário de contato validado em JavaScript, que salva informações no localStorage e fornece feedback visual. A tipografia Montserrat, paleta em tons terrosos e imagens de alta qualidade reforçam o conceito acolhedor e artesanal da marca.

🎨 Site do Atelier

Projeto fictício desenvolvido como uma landing page para um atelier especializado em bolos, doces, salgados e kits para festas. O foco foi criar uma experiência visual envolvente e intuitiva, utilizando HTML5 semântico e CSS3 responsivo para garantir perfeita adaptação em diferentes dispositivos. O site apresenta vitrine de produtos organizada por seções, menu interativo para navegação rápida, área de contato e redes sociais, e um design que valoriza a identidade visual do atelier. A proposta é transmitir sofisticação e clareza, destacando os produtos de forma atraente e funcional.

💰 API de Criptomoeda

Aplicação web interativa que apresenta cotações atualizadas das principais criptomoedas, com dados obtidos em tempo real por meio de integração com API pública. O projeto foi desenvolvido com HTML, CSS e JavaScript puro, destacando habilidades em consumo de APIs, manipulação dinâmica do DOM e criação de interfaces responsivas. Ideal para acompanhar tendências do mercado cripto de forma simples e intuitiva.
